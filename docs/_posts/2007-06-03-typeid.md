Visual C++ .NET下，如果/GR 未打开，这个程序仍然成功打印.

但如果 public: virtual void vvfunc(), 则 typeid( *pb )， typeid( *pd )的会错误，抛出__non_rtti_object异常。

如果上面注释的两句打开，则会跑出__bad_typeid异常。

__non_rtti_object继承自__bad_typeid.
