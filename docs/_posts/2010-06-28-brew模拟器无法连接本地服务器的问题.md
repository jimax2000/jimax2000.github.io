<p>BREW模拟器无法连接本地服务器的问题</p> <p>Edit the \"Sockets\" entry in your device skin file. <br/>The entry has the following structure BREW2.1 onwards. <br/><br/>Sockets &lt;nSocketLimit&gt; &lt;bListenAccept&gt; &lt;bShutdown&gt; &lt;bLoopback&gt; &lt;nMaxSendTo&gt; <br/><br/>where the new members are -&gt; <br/><br/>bListenAccept: the device supports Listen(). Default: 0 <br/>bShutdown: the device supports Shutdown(). Default: 0. <br/>bLoopback: the device supports a loopback interface (i.e. 127.0.0.1). Default: 0. <br/>nMaxSendTo: the maximum payload size for UDP datagrams sent via OEM_SendTo(). Default: 1472 bytes.</p> <p>主要是Loopback一项，改为 1</p> <p><br/><br/></p>
